# Overview

This is a personal portfolio website built using HTML and CSS to showcase projects, skills, and contact information. The website features a clean, responsive design suitable for displaying professional work and achievements.

# Features

1. Responsive Design: Adapts to various screen sizes, including mobile, tablet, and desktop.

2. Navigation Bar: Easy access to Home, About, Projects, and Contact sections.

3. Project Showcase: Displays projects with descriptions and links.

4. Contact Section: Includes links to social media and a contact form.

5. Modern Styling: Uses CSS for visually appealing layouts, hover effects, and animations.

# Technologies Used
1. HTML5: For semantic structure and content organization.

2. CSS3: For styling, layouts, and responsive design (including Flexbox and Grid).
